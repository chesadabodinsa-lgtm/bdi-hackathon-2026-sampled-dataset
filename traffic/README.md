# ชุดข้อมูลภาพกล้องจราจรเมืองขอนแก่น

## คำอธิบาย
ชุดข้อมูลนี้ประกอบด้วย 2 ส่วนหลัก:
1. **วิดีโอจากกล้องตรวจจราจร** จำนวน 35 คลิปจาก YouTube  มุ่งเน้นโจทย์การนับยานพาหนะแบบไม่ซ้ำ (Unique Vehicle Counting)
2. **ข้อมูลสถิติการจราจรแบบตาราง** จำนวน 25 จุด จาก Khon Kaen Traffic Dashboard  เป็นข้อมูลนับยานพาหนะแยกประเภท (รถยนต์/รถจักรยานยนต์/รถบรรทุก) พร้อมพิกัด GPS

แหล่งข้อมูลวิดีโอ: [YouTube Playlist](https://www.youtube.com/playlist?list=PLTtSg8Z-ZIjopXh1ajC-Mx96Pp6xj-cjV)
แหล่งข้อมูลสถิติ: [KK Traffic Dashboard](https://suchartstudio.github.io/kktraffic/)

## ฟิลด์ข้อมูล

### วิดีโอกล้องจราจร
- วิดีโอจากกล้องตรวจจราจรบันทึกภาพการจราจรในมุมสูง (overhead view) ครอบคลุมจุดตัดและถนนสำคัญในเขตเมือง

### ข้อมูลสถิติการจราจร (ตาราง 25 แถว × 9 คอลัมน์)
| ฟิลด์ | ประเภท | รายละเอียด |
|------|--------|----------|
| `ที่` | `int` | ลำดับจุดตรวจ (1–25) |
| `เส้นทาง` | `string` | ชื่อถนน (เช่น ถ.กสิกรทุ่งสร้าง) |
| `ตำแหน่งติดตั้งเครื่องวัด` | `string` | ตำแหน่งกล้อง (เช่น หน้าตลาดจอมพล) |
| `Lat` | `float` | ละติจูด (~16.37–16.49) |
| `Lag` | `float` | ลองจิจูด (~102.78–102.85) |
| `Car` | `int` | จำนวนรถยนต์นับได้ (18,913–66,220) |
| `Motorcycle` | `int` | จำนวนรถจักรยานยนต์นับได้ (1,667–47,205) |
| `Truck` | `int` | จำนวนรถบรรทุกนับได้ (2,681–19,043) |
| `รวมต่อวัน` | `int` | จำนวนยานพาหนะรวมต่อวัน (53,419–160,892) |
| `คัน/ชั่วโมง` | `int` | อัตราเฉลี่ยต่อชั่วโมง (2,755–8,298) |

> **หมายเหตุ**: ข้อมูลสถิติเป็นนับฝั่งเดียว (single-lane count) จากเครื่องวัดจราจร

## โครงสร้างไฟล์
| ไฟล์ | รายละเอียด |
|------|----------|
| (วิดีโอจาก YouTube) | คลิปวิดีโอกล้องจราจร 35 คลิป |
| (ข้อมูลจาก Dashboard) | สถิติ 25 จุดตรวจ พร้อมพิกัด GPS และจำนวนยานพาหนะ |

## โจทย์ / แนวทางวิเคราะห์
- **Unique Vehicle Counting**: นับจำนวนยานพาหนะที่ผ่านจุดตรวจโดยไม่ซ้ำ (รถคันเดียวกันที่ผ่านหลายครั้งนับเป็น 1)
- **Vehicle Classification**: จำแนกประเภทยานพาหนะ (รถยนต์, รถจักรยานยนต์, รถบรรทุก, รถ buses)
- **Traffic Flow Analysis**: วิเคราะห์ความหนาแน่นการจราจรตามช่วงเวลา (peak/off-peak)
- **Intersection Performance**: ประเมินประสิทธิภาพของจุดตัดการจราจร
- **Spatial Analysis**: วิเคราะห์รูปแบบการจราจรตามพื้นที่ (ใช้พิกัด GPS 25 จุด)
- **Cross-validation**: เปรียบเทียบผลนับจากวิดีโอกับข้อมูลสถิติจากเครื่องวัด

## หมายเหตุ
- วิดีโอทั้งหมดมาจากแหล่ง YouTube เดียวกัน (35 คลิป)
- ข้อมูลสถิติมาจาก KK Traffic Dashboard (25 จุดตรวจ)
- ข้อมูลสถิติเป็นการนับฝั่งเดียว (single-lane) ไม่ใช่ทั้งสองฝั่งถนน
