# ชุดข้อมูลบันทึกคำร้องจากประชาชน เทศบาลนครขอนแก่น

## คำอธิบาย
ชุดข้อมูลคำร้อง/ร้องเรียนของประชาชนต่อเทศบาลนครขอนแก่น จำนวนประมาณ **46,000 รายการ** (ตัวอย่าง **462 รายการ**) ข้อมูลบันทึกประวัติคำร้องตั้งแต่วันที่ได้รับจนกระทั่งดำเนินการเสร็จสิ้น

## ฟิลด์ข้อมูล
| ฟิลด์ | คำอธิบาย |
|-------|----------|
| `ส่วนงาน` | ส่วนงานที่รับผิดชอบคำร้อง |
| `ฝ่าย` | ฝ่ายภายในส่วนงาน |
| `เลขคำร้อง` | เลขที่อ้างอิงคำร้อง |
| `เรื่องร้องทุกข์` | หัวข้อ/ประเด็นคำร้อง |
| `ประเภทคำร้อง` | ประเภทของคำร้อง |
| `เขต` | เขตพื้นที่ของเทศบาล |
| `ชุมชน` | ชุมชน/พื้นที่ที่เกิดเหตุ |
| `วันที่รับเรื่อง` | วันที่เทศบาลรับคำร้อง |
| `วันที่เสร็จ` | วันที่ดำเนินการเสร็จสิ้น |
| `สถานะ` | สถานะปัจจุบันของคำร้อง |

## โจทย์ / แนวทางวิเคราะห์
- **Text Classification**: จัดหมวดหมู่คำร้องอัตโนมัติจากเนื้อหา
- **NLP / Topic Modeling**: ประเด็นร้อนที่ประชาชนร้องเรียนบ่อย
- **Performance Analytics**: วิเคราะห์ระยะเวลาในการดำเนินการ (รับ → เสร็จ)
- **Geospatial Analysis**: แผนที่ความหนาแน่นคำร้องตามเขต/ชุมชน
- **Predictive Model**: พยากรณ์วันเสร็จสิ้น หรือระบุส่วนงานที่เหมาะสม
